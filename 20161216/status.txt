Что сделано за неделю

=====================

Занимались:

- получили baseline для работы с Байкал-Т и Таволгой:

- комплект для сборки приложений (SDK+rootfs);

            - инструкцию/последовательность сборки, которая применялась для текущей версии chromium;

- драйвер для SM750 с историей изменений в fbturbo;

- изучали возможные варианты тестирования производительности iceweasel и chromium

- в соответствии с инструкцией по сборке chromium, начали компилирование toolchain с помощью crosstool. Обнаружили проблемы с ISL и gload,

- устанавливали build-сервер на стороне MERA для возможности непрерывной интеграции изменений в пакет chromium (и дистрибутив Debian, при необходимости)



Сделали:

- собрали результаты Dromaeo и Octane 2.0 с Таволги и двух других систем на Intel Atom. Сводная таблица по Dromaeo приложена к отчету. Результаты по Octane 2.0 повторно соберем в понедельник, т.к. требуется перзапуск браузера после каждой итерации (занимает время).

- для исследования производительности "чистой системы", собрали дистрибутив на последних доступных инструментах gcc-6.2 и binutils-2.27, с интеграцией ядра для Baikal и пакета Chromium. Планируем провести тестирование на следующей неделе.

- прояснили вопросы с историей SM750, просмотрели изменения для fbturbo, начали изучение вариантов возможной оптимизации.

- завершить установку build-сервера;

- собрать версию chromium с последней доступной версией исходных кодов; измерить производительность полученной версии;

- провести анализ узких мест при использовании JavaScript, по возможности пересобрать chromium с новыми настройками/ключами оптимизации.
